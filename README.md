# GreenLedger 🌿

GreenLedger is a modern, professional, and sustainable expense tracking application designed to help users manage their finances efficiently with a premium user experience.

## ✨ Features

- **Professional Dashboard**: Real-time summary of expenses and financial status.
- **Advanced Expense Management**: Easily add, edit, and categorize expenses.
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Interactive Components**: Built-in calculator, dynamic charts, and smooth animations.
- **State Management**: Robust state handling using NgRx.
- **Security**: Mock authentication and Firebase integration options.
- **Data Export**: Export your expense reports to PDF.

## 🚀 Tech Stack

- **Framework**: [Angular 18](https://angular.io/)
- **State Management**: [NgRx](https://ngrx.io/)
- **Styling**: Vanilla CSS with modern design principles (Glassmorphism, Vibrant Palettes).
- **Database/Auth**: [Firebase](https://firebase.google.com/)
- **Utility Libraries**: Dexie.js (IndexedDB), CryptoJS, jsPDF, RxJS.

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [Angular CLI](https://angular.io/cli)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/TitanXS75/Greenledger.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Greenledger
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

To start the development server:
```bash
npm run dev
```
Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## 📦 Build

To build the project for production:
```bash
npm run build
```
The build artifacts will be stored in the `dist/` directory.

## 🧪 Testing

- **Unit Tests**: `npm run test`
- **E2E Tests**: `npm run e2e`
